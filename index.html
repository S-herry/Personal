作品1

這個網站專案旨在介紹台灣苗栗縣的熱門景點、美食特色以及周邊交通資訊。透過整合相關資訊，方便旅客快速掌握苗栗的魅力，並利於其規劃前往的旅遊行程。
網站的設計強調了使用者體驗和互動性，採用了以下技術： Bootstrap：用於響應式設計，確保網站能夠在各種裝置上保持良好的外觀和操作體驗。 CSS/SCSS：透過自訂樣式來強化頁面視覺設計，增強整體一致性和美感。 HTML：搭建網頁的基本結構，確保內容清晰易讀。 JavaScript：提供互動功能，像是景點介紹的動態展示和選單的即時切換。 這個網站首頁展示了多個核心區塊： 熱門景點地圖：以直觀的方式呈現苗栗各大景點的地理位置，並支援快速篩選功能。 美食推薦：介紹在地特色餐廳和小吃，吸引美食愛好者。 交通資訊：提供周邊交通動線，協助遊客輕鬆規劃行程。 此外，設計上還包含了互動功能，例如點擊地圖標記能即時顯示該景點的詳細介紹、相片及評分。整體來說，這個網站不僅展示了苗栗的觀光亮點，還兼具實用性，讓使用者能快速找到所需資訊，提升了瀏覽體驗。

作品2

Vege 蔬果箱販售訂閱服務 生活節奏快速的上班族常常無暇規劃購買新鮮蔬果的時間。在繁忙的超市採購過程中，我們可能買了太多，導致蔬果過期浪費。或者你可能不了解蔬果的特性，不確定應該如何挑選適合自己的品項。為了解決這些問題，我們設計了一套便捷的消費者喜好檢測方法，並提供定期定額配送蔬果箱的服務，讓消費者能夠在線上快速完成客製化訂購。 在這次專案中，我負責產品資訊頁面和診斷功能的開發，這讓消費者可以簡單了解蔬果的寒性、平性、熱性特徵，並根據自身需求選擇適合的產品。我們使用 Node.js 連結 MySQL 資料庫，打造了具有良好互動性的產品資訊頁面。整體設計以及互動功能的實現基本都使用 jQuery 函式庫和 Bootstrap 框架，為使用者帶來流暢且愉悅的網站體驗。
